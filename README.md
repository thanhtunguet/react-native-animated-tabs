# react-native-animated-tabs

Animated tabs for React Native

## Installation

```sh
npm install react-native-animated-tabs
```

## Usage

```js
import AnimatedTabs from "react-native-animated-tabs";

// ...

const result = await AnimatedTabs.multiply(3, 7);
```
